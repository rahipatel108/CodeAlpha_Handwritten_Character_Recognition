# Handwritten Character Recognition

## Objective
To recognize handwritten digits using Convolutional Neural Networks (CNN) and the MNIST dataset.

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Dataset
MNIST Dataset (60,000 training images and 10,000 testing images)

## Model Architecture
- Conv2D Layer
- MaxPooling2D Layer
- Flatten Layer
- Dense Layer
- Softmax Output Layer

## Features

- Handwritten digit recognition
- CNN-based digit classification
- Model evaluation
- Prediction on unseen digits
  
## Results
- Training Accuracy: 98.97%
- Test Accuracy: 98.03%
